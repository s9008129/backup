📞 常見問題 & 解決方案
═══════════════════════════════════════════════════════════════════════════════

Q: setup_all.bat 無法執行？
A: • 檢查 Python 是否已安裝: python --version
   • 檢查 Conda 是否已安裝: conda --version
   • 以管理員身份執行命令提示符

Q: 環境驗證失敗？
A: • 執行 conda activate backup
   • 執行 python validate_env.py --verbose
   • 查看錯誤訊息並重新運行 setup_all.bat

Q: Git 無法初始化？
A: • 檢查 Git 是否已安裝: git --version
   • 確認資料夾有寫入權限
   • 執行 python init_git.py (Python 版本)

Q: 應用無法啟動？
A: • 確認環境已激活: conda activate backup
   • 檢查 tkinter: python -c "import tkinter; print('OK')"
   • 查看控制台錯誤訊息
   • 執行 python backup_tool.py 2>&1 | tee error.log

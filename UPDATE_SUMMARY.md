🚀 立即開始
════════════════════════════════════════════════════════════════════════════════

1️⃣ 驗證安裝

   python --version              # Python 3.11+
   conda --version               # Conda 23.x+
   git --version                 # Git 2.x+

2️⃣ 建立 Conda 環境

   cd D:\dev\backup
   conda create -n backup python=3.11 -y
   conda activate backup

3️⃣ 安裝依賴

   pip install -r requirements.txt

4️⃣ 驗證完成

   python validate_env.py

5️⃣ 啟動應用

   python backup_tool.py

💾 Git 操作指南
════════════════════════════════════════════════════════════════════════════════

在 Copilot CLI 中使用自然語言進行版本控制:

查看狀態:
  👉 "show me git status" 或 "git status"

檢查變更:
  👉 "show me what changed" 或 "git diff"

暫存檔案:
  👉 "add all files" 或 "git add ."

提交變更:
  👉 "commit with message: 修復問題" 或 "git commit -m 'message'"

查看日誌:
  👉 "show recent commits" 或 "git log --oneline -5"

查看詳細提交:
  👉 "show latest commit details" 或 "git show HEAD"
